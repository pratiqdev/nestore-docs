# Using Nestore

Setup is super easy - create, export and use your store. 

## Create a Store

Import nestore and initialize with a store - an object containing your state

```jsx title="myStore.js"
import nestore from '@pratiq/nestore'

const nst = nestore({

  logged_in: false,
  user: null,
  messages: [],

  login: (NST, [name, password]) => {
    NST.set('logged_in', true)
    NST.store.user = name
  }

})

export default nst
```






## Import the Store

Then import your store, register listeners on any path, and interact with the store

```jsx title="app.js"
import nst from './myStore.js'

nst.on('user', ({ key, path, value }) => {
    console.log(`Logged in as ${value}`)
})

nst.login('Alice', '1234')
nst.set('messages')
```







See **[the examples](/docs/category/examples)** for a better look at setting up your store in different environments
